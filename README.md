# Physics Sims

A modular **C++ physics simulation framework** for exploring complex physical systems such as black holes, electron clouds, and atomic nuclei.  
This project focuses on high-performance numerical simulation, modular design, and clear visualization of physical processes.

---

## Overview

This repository contains a collection of physics simulations built around a shared core engine.  
Each simulation implements its own physical model while reusing common numerical tools and infrastructure.

Planned simulations include:

- **Black Hole Simulation**
  - Gravitational dynamics
  - Orbital motion
  - Relativistic effects (future work)

- **Electron Cloud Simulation**
  - Probability density visualization
  - Quantum orbital approximations
  - Electron distribution modeling

- **Atomic Nucleus Simulation**
  - Nucleon interactions
  - Nuclear structure models
  - Particle dynamics

The goal of this project is to build a general simulation framework capable of modeling systems from astrophysics to quantum physics.

---


## Goals

- Build a high-performance simulation engine in C++
- Create modular physics simulations
- Visualize physical systems across multiple scales
- Maintain a clean and extensible architecture
